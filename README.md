Gemini APIを使用して、RSSからニュースを動的に取得・分類するWebアプリケーションです。

システムの仕組み（ロジック）

1. **フロントエンド (index.html)**
   - ユーザーがプルダウンメニューで「カテゴリ（政治・IT・スポーツ）」を選択します。
   - Vue.jsが選択されたカテゴリを検知し、バックエンドAPIに対してリクエストを送信します。

2. **バックエンド (Python/Flask)**
   - フロントエンドからのリクエストを受け取ります。
   - 選択されたカテゴリに対応するRSSフィード（NHK、Yahooなど）から最新ニュースを取得します。
   - 取得したニュースのタイトルと概要をGemini APIに送信し、AIが内容に適した「タグ」を生成します。
   - ニュースデータと生成されたタグをJSON形式でフロントエンドに返却します。

3. **表示**
   - フロントエンドがJSONデータを受け取り、ニュース一覧とタグを画面に描画します。

起動手順

このアプリケーションは、バックエンドをGoogle Colab上で実行し、フロントエンドをローカルブラウザで開く構成になっています。

1. バックエンドの起動
1. `app.py` のコードをGoogle Colabにコピーします。
2. コード内の `YOUR_NGROK_TOKEN` と `YOUR_GEMINI_API_KEY` を自身のキーに書き換えます。
3. セルを実行し、生成された `ngrok` のURL（例: `https://xxxx.ngrok-free.app`）をコピーします。

2. フロントエンドの起動
1. `index.html` をテキストエディタで開きます。
2. `const BACKEND_URL = "..."` の部分に、先ほどコピーしたURLを貼り付けます。
3. `index.html` をブラウザで開くとアプリが動作します。

 使用技術
- Frontend: HTML, CSS, Vue.js
- Backend: Python, Flask, Google Gemini API
